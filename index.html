CSP Coding

CSP

CSP from "Central Software Programming language", is a new programming language created to make it not only easier for programmers to work in user applications, but give a lot of more features, that old languages could not have done it. Its use variants between user applications, web development, game development or more. It is easy to learn and use and gives access from the user directly into the hardware.

GSML

GSML is a modified design language running through csp coding. It is useful for graphic and animated layout design / development. It uses similar syntacts as csp, html and css. But comes with more abilities and features to do.

Editor

The Editor is the compiler of CSP. CSP is an open source language, therefore making it available for all clients to create their own framework on csp itself. Mayor frameworks like csp+, Novarel etc. are fully created on CSP. The main editor is CSPeditor. The main purpose of it, is for software applications. But there are more...
